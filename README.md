## Projects

### 🛒 Project 1 — Supermarket Sales EDA
Exploratory data analysis on 50 transactions across Bangkok, Chiang Mai and Phuket.

**Key findings:**
- Electronics generated the most revenue at $1,007
- Chiang Mai customers gave the highest average rating (8.23)
- Credit Card and E-wallet tied as most popular payment method

**Skills:** pandas, head, info, describe, groupby, value_counts

---

### 🧹 Project 2 — Data Cleaning
Cleaned a dirty version of the supermarket dataset containing missing values and duplicates.

**What I did:**
- Removed 3 duplicate rows
- Filled missing numeric values with column mean
- Filled missing text values with 'Unknown'

**Skills:** isnull, dropna, fillna, drop_duplicates

---

### 🎬 Project 3 — Movie Industry Analysis (1972–2022)
Analyzed 50 movies to find patterns in revenue, ratings, and profitability.

**Key findings:**
- Action genre dominates with $17,741M total revenue
- Budget does not guarantee higher ratings — The Godfather rated 9.2 on a $6M budget
- Frank Darabont is the highest rated director at 9.3 average
- Avengers Endgame most profitable at $2,442M profit

**Skills:** feature engineering, groupby, scatter plot, bar chart, line chart, annotate, nlargest
